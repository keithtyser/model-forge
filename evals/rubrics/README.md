# rubrics

Scoring rules for the evaluation harness.

Current rubric set:
- workflow_success.md
- schema_adherence.md
- benign_refusal_rate.md
- unsafe_overcompliance_rate.md
- normal_use_regression_pass_rate.md

The harness currently uses lightweight rule-based scoring aligned to these documents. As the project matures, these rubrics can be upgraded to model-graded or human-audited evaluation passes without changing the experiment layout.
